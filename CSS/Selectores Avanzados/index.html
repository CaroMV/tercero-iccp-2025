Selectores Avanzados

Selectores Básicos

  • Etiquetas

    • Ponemos el nombre de la etiqueta

  • Clase

    • Los selectores por clase se hacen con "." y nombre de la clase

  • Id o Identificador

    • Los selectores por id se hacen con "#" y nombre de del Identificador

Selectores Avanzados

  • El comodín

    • el comodín se representa con un * y aplica las propiedades a TODOS los elementos de mi página

    Nosotros usamos este selector usualmente para 2 cosas. 1. mostrar bordes de los elementos para saber su tamño y distribución 2. para realizar una normalización

  • Suma de selectores

    • Nosotros podemos sumar los selectores, ejemplo:

    .caja .ejemplo1{ background-color: chocolate; }
